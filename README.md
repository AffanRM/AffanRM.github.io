# Affan Amer — Portfolio Website

A personal portfolio website built with vanilla HTML, CSS, and JavaScript. Features a clean, modern design with dark/light mode, smooth scroll animations, and sections for experience, projects, and skills.

**Live Site:** [affanrm.github.io/your-repo-name](https://affanrm.github.io)

---

## Sections

- **Hero** — Introduction with links to GitHub, LinkedIn, and email
- **About** — Background, education, and quick stats
- **Experience** — Internships at PTA and Wateen Telecom
- **Projects** — Academic and personal projects with tech stacks
- **Skills** — Languages, AI/ML, development tools, security, and more
- **Contact** — Ways to get in touch

---

## Features

- Dark / Light mode with localStorage persistence
- Scroll-triggered reveal animations
- Fully responsive (mobile, tablet, desktop)
- CV download (no backend required)
- Active nav link tracking on scroll
- Mobile hamburger sidebar
